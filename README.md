## Descripción

[Curso](https://www.udemy.com/course/nest-framework/) de NestJs realizado por el profesor Fernando Herrera desde la plataforma de [Udemy](https://www.udemy.com).

## Instalación

```bash
$ pnpm install
```

## Ejecutando la app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
